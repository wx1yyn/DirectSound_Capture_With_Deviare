Deviare Direct Sound Capturing Sample
=====================================

This is a python sample on how to capture DirectSound audio using Deviare. Enjoy!


Dependencies
==========

* Deviare (http://www.nektra.com/products/deviare/download.php)
* Python +2.x
* Python Extensions for Windows (http://sourceforge.net/projects/pywin32)

RUN
===

* Execute 'run_me.py'
